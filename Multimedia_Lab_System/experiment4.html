实验 4：多媒体元素应用

掌握音频、视频、图像插入及超链接、锚记的使用

一、实验目的

  • 熟悉 Dreamweaver 或其他网页编辑器的核心操作
  • 掌握网页中文本格式的精细化编辑技巧
  • 精通多媒体元素（音频、视频、图像）的插入及格式自定义
  • 掌握文本超链接、图像热点超链接的创建与优化
  • 学会锚记命名及锚链接的创建，提升页面导航体验

二、实验内容及步骤

1. 站点创建与规划

按照规范创建实验站点文件夹，建立清晰的目录结构：根文件夹下包含 html 文件、css 文件夹（存放样式文件）、media 文件夹（分 images/audio/video 子文件夹存放媒体文件），确保文件路径统一，避免加载错误。

2. 文本格式编辑

使用 HTML 基础标签结合 CSS 实现文本美化，html示例代码如下： 


<h2 class="sub-title">文本格式示例</h2>
<p class="text-content">
    这是一段标准文本，通过 CSS 统一控制字体、颜色和行高。
</p>

对应的css示例代码如下： 


/* 子标题样式 */
.sub-title {
    font-size: 20px;
    font-weight: 600;
    color: #1d1d1f;
    margin-bottom: 16px;
    line-height: 1.4;
}

/* 正文内容样式 */
.text-content {
    font-size: 16px;
    color: #424245;
    line-height: 1.8;
    margin-bottom: 20px;
}

3. 多媒体元素插入

3.1 图像插入与优化

使用 img 标签插入图像，设置圆角、阴影提升视觉效果，代码示例：


<h2>3.1 图像插入与优化</h2>
<div class="media-container">
    <p class="media-desc">带圆角和精致阴影的校徽展示：</p>
    <img src="media/images/校徽.png" alt="校徽" width="250" height="250">
</div>

3.2 视频插入（两种核心方式）

方式1：embed 标签（兼容旧浏览器）


<div class="media-container">
    <p class="media-desc">基于 embed 标签的视频播放（兼容旧浏览器）</p>
    <embed src="media/video/AI.mp4" type="video/mp4" width="800" height="450">
</div>

方式2：video 标签（HTML5 推荐，支持控件自定义）


<div class="media-container">
    <p class="media-desc">基于 HTML5 video 标签的视频播放（推荐）</p>
    <video width="800" height="450" controls="controls">
        <source src="media/video/AI.mp4" type="video/mp4">
        您的浏览器不支持 HTML5 视频播放，请升级至 Chrome、Edge、Firefox 等现代浏览器
    </video>
</div>

效果展示：

3.3 音频插入（两种核心方式）

方式1：embed 标签


<div class="media-container">
    <p class="media-desc">基于 embed 标签的音频播放</p>
    <embed src="media/audio/bg-music.mp3" type="audio/mpeg" width="600" height="45">
</div>

方式2：audio 标签（支持循环播放）


<div class="media-container">
    <p class="media-desc">基于 HTML5 audio 标签的音频播放（支持循环）</p>
    <audio controls="controls" loop="loop">
        <source src="media/audio/bg-music.mp3" type="audio/mpeg">
    </audio>
</div>

4. 超链接创建

  • 文本超链接：<a href="index.html" class="link">返回首页</a>（添加 hover 下划线动画，具体css文件可以自行设计）
  • 图像热点超链接：给图像添加可点击区域，链接到不同实验页面，代码示例： 

<img src="media/images/校徽.png" alt="校徽" usemap="#expMap" width="800" style="border-radius: 8px;">
<map name="expMap">
    <!-- 矩形热点：链接到首页 -->
    <area shape="rect" coords="100,100,250,250" href="index.html" alt="index"
          onmouseover="this.style.cursor='pointer'">
</map>

具体效果：点击左上方校徽即可跳转首页
